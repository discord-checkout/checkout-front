# Checkout Frontend

> **Checkouts your wardrobe** — AI 기반 퍼스널 패션 솔루션으로 텅 빈 옷장을 채워드립니다.

🔗 **[라이브 데모 보러가기](https://checkout-front-rho.vercel.app/)**

---

## 서비스 소개

결국 돌고 돌아 입는 건 옷장의 체크셔츠. 돈도 없고 시간도 없고 실패하기는 두려운 사람들의 포기 선언문이죠.

**Checkout은 "무엇을 사야 옷장이 굴러가는가?"** 라는 질문에서 출발합니다. 이미 옷이 많은 사람을 위한 코디 앱이 아닌, 텅 빈 옷장을 전략적으로 채워가는 생존형 패션 가이드입니다.

---

## 핵심 기능

### 1. 3초 컷 스타일 진단 (온보딩)
옷 사진 없이 스타일 카드 스와이프 + 클릭 몇 번으로 취향을 입력하면, AI가 스타일 · 핏 · 라이프스타일 · 예산을 즉시 정의합니다.

### 2. 첫 번째 옷 핀셋 추천 (홈)
진단 결과를 바탕으로 지금 당장 옷장의 기둥이 될 아이템 단 하나를 추천합니다. 단순한 추천을 넘어, **왜 이 옷이어야 하는지** 근거와 코디 조합까지 함께 제시합니다.

### 3. 단계별 옷장 로드맵 (로드맵)
매월 정해진 예산 안에서 아이템을 하나씩 추가하며 코디 조합이 기하급수적으로 늘어나는 **1 · 2 · 3개월 구매 로드맵**을 제공합니다.

## 인증 방식

모든 API 요청에 `X-User-ID` 헤더(UUID)를 포함해 사용자를 식별합니다.  
UUID는 `localStorage`에 저장되며, 온보딩 시작 시 새로 발급됩니다.
